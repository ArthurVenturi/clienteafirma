Plugins

Junto com o Autofirma são distribuídos diversos plugins que permitem adicionar funções adicionais ao aplicativo.

Cálculo e verificação de hashes

Este plugin adiciona ao Autofirma opções para calcular o hash de um arquivo ou de todos os arquivos de um diretório e verificar esses mesmos hashes que foram gerados.

Uma vez instalado o plugin, aparecerá na barra de menus do Autofirma a opção "Hashes digitais" com as diferentes operações que podem ser realizadas.

  • Calcular hash digital: Permite calcular o hash de um arquivo e salvá-lo em disco. O formato e a extensão do arquivo de hash variarão conforme o formato selecionado.

    • Arquivo para calcular o hash: Arquivo do qual desejamos calcular o hash.
    • Algoritmo de hash: Algoritmo de hash.
    • Formato do hash: Codificação utilizada para salvar o hash:
      • Hexadecimal em ASCII (Base16): O hash será armazenado como uma cadeia hexadecimal terminada em 'h'.
      • Base64: O hash será armazenado em Base 64.
      • Binário: O hash será armazenado sem codificação.
    • Copiar hash para a área de transferência: Ao ativar esta opção, além de salvar o hash em arquivo, ele será copiado para a área de transferência. O hash será copiado com a codificação do formato selecionado ou, se o formato binário for selecionado, em hexadecimal.

  • Verificar hash digital: Permite verificar o hash de um arquivo.

    • Arquivo de dados: Arquivo ao qual corresponde o hash que se deseja verificar.
    • Arquivo que contém o hash: Arquivo com o hash.

  • Calcular hash digital em diretório: Permite calcular o hash dos arquivos de um diretório.

    • Diretório para calcular os hashes: Diretório com os arquivos dos quais queremos calcular o hash.
    • Algoritmo de hash: Algoritmo de hash.
    • Recursivo: Ao ativar, também será calculado o hash dos arquivos localizados nos subdiretórios do diretório selecionado.

  • Verificar hash digital em diretório: Permite verificar o hash dos arquivos de um diretório.

    • Diretório: Diretório com os arquivos aos quais correspondem os hashes que se deseja verificar.
    • Arquivo de hashes: Arquivo com os hashes que serão verificados.

Ao instalar o plugin de hashes digitais em sistemas Windows, também são adicionadas as opções de gerar e verificar hashes digitais no menu de contexto de arquivos e diretórios do sistema.

As opções que aparecem são:

  • Gerar hash digital: Esta opção aparece no menu de contexto de todos os arquivos e diretórios e permite gerar o hash do arquivo ou dos arquivos do diretório.
  • Verificar hash digital: Esta opção aparece no menu de contexto dos arquivos de hash (.hash, .hashb64, .hashfiles e .hexhash) e permite selecionar o arquivo ou diretório do qual se deseja verificar o hash.
