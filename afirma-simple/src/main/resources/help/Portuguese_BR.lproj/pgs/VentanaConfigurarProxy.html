Configuração de proxy

O Autofirma precisa acessar a Internet por diferentes motivos e, se necessário, deve-se configurar um proxy de rede para permitir isso. Os propósitos para os quais o Autofirma pode requerer acesso à rede são:

  • Comunicar-se com páginas web onde são realizados procedimentos de assinatura.
  • Buscar atualizações (apenas se a opção estiver habilitada).
  • Enviar estatísticas (apenas se a opção estiver habilitada).

Todas as conexões de rede realizadas pelo Autofirma aplicarão a configuração de proxy aqui estabelecida.

As opções de configuração de proxy são:
  • Sem proxy: As conexões de rede serão feitas diretamente.
  • Configuração de proxy do sistema: O aplicativo identificará a configuração que deve aplicar para cada URL conforme a configuração estabelecida no sistema operacional e/ou navegador. Esta configuração pode não ser exatamente igual à estabelecida no Windows.
  • Configuração manual de proxy: Permite definir a configuração específica de proxy a ser utilizada. As opções configuráveis são:
    • Host: Nome de domínio ou IP do proxy.
    • Porta: Número da porta de acesso ao proxy.
    • Usuário: Nome de usuário a ser utilizado quando o proxy exigir autenticação. Opcional.
    • Senha: Senha do usuário a ser utilizada quando o proxy exigir autenticação. Opcional.
    • Não usar proxy para os seguintes endereços IP e domínios: Lista de endereços IP e nomes de domínio que devem ser acessados diretamente (sem passar pelo proxy configurado manualmente). Comumente, são configurados aqui os endereços da intranet quando operações de assinatura são realizadas a partir de aplicações web situadas tanto dentro quanto fora da sua intranet.
      Os endereços devem ser separados por vírgulas (,) e podem ser usados asteriscos (*) no início ou fim de cada um como coringa para identificar subdomínios ou sub-redes completas. Opcional.

    Adicionalmente, são fornecidas as seguintes opções:

    • Autodetectar: Extrai da configuração do sistema o host e a porta a serem utilizados para um endereço genérico.
    • Verificar conexão: Verifica a configuração manual de proxy para o acesso a um endereço genérico.

A configuração estabelecida será aplicada ao clicar no botão OK. Se clicar no botão Cancelar, a janela será fechada sem aplicar a configuração.
