Certificados de confiança

O Autofirma realiza conexões com servidores remotos para diversas funções. Nesta seção, é possível importar para o repositório de confiança do Autofirma os certificados dos servidores aos quais o Autofirma se conecta para que sejam reconhecidos como confiáveis.

As opções nesta tela são:
  • Importar: Ao selecionar esta opção, será aberta uma janela onde é possível importar um certificado selecionando o arquivo de uma rota local ou de uma URL. No caso de importação local, é possível selecionar um ou mais arquivos e, no caso de importação por URL, serão importados o emissor e seus certificados superiores da cadeia de certificação.

    Uma vez obtidos os certificados, aparecerá a seguinte tela com os certificados carregados.

    As opções nesta janela são:

    • Ver certificado: Abre uma janela com as informações do certificado selecionado na lista.
    • Omitir: Descarta o certificado selecionado na lista para que não seja importado.
    • Importar certificados: Importa os certificados indicados na lista para o repositório de confiança do Autofirma.
    • Fechar: Fecha a janela.

  • Ver certificado: Abre uma janela com as informações do certificado selecionado na lista.
  • Excluir: Permite excluir do repositório de confiança do Autofirma o certificado selecionado.

Ao clicar no botão Aceitar, a janela será fechada.
