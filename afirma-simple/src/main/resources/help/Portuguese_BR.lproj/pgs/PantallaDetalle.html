Tela de detalhes da assinatura ou assinaturas realizadas

Após finalizar uma operação de assinatura, é exibida uma tela com informações sobre o resultado da operação.

Se desejar obter mais informações sobre a assinatura, pode clicar no link Detalhes da assinatura e será exibido um diálogo com informações sobre ela:

Esta tela se divide em três seções:

Informações sobre o arquivo assinado ou diretório de saída
É exibido o caminho completo do arquivo assinado (pode copiar este caminho clicando com o botão direito do mouse) ou do diretório de saída. Se foi realizada a assinatura PAdES de um documento PDF, será exibido um botão para abrir o documento assinado com o aplicativo padrão do sistema.

Informações sobre o certificado utilizado para realizar as assinaturas
É possível ver o certificado (pelo aplicativo padrão do sistema operacional) clicando nos hyperlinks.

Informações detalhadas sobre a assinatura ou conjunto de assinaturas realizadas
No caso de assinar um documento, são exibidos pelo menos os seguintes pontos em uma estrutura de árvore:

  • Formato da assinatura: CAdES, PAdES, XAdES, Factura-e, OOXML ou ODF.
  • Dados assinados: Indica se os dados estão embutidos na própria assinatura e permite acessá-los caso estejam.
  • Árvore de assinaturas: Estrutura hierárquica das assinaturas encontradas no documento. É possível ver o certificado usado para cada uma delas clicando no respectivo link.

Para navegar entre os elementos da árvore, podem ser usadas as setas do teclado.

Caso tenha assinado múltiplos arquivos, será exibida uma lista com os arquivos gerados, seu tamanho e resultado. No caso das assinaturas que não puderam ser geradas, será exibido o caminho do arquivo de entrada. É possível dar duplo clique em uma das assinaturas da lista para acessar suas informações detalhadas.
