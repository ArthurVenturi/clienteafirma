Propriedades das assinaturas visíveis em PDF

Ao habilitar a criação de assinaturas visíveis em PDF, o usuário poderá selecionar uma área de assinatura sobre o PDF que será assinado no formato PAdES.

Este menu permite usar o mouse para selecionar uma área retangular de uma das páginas do PDF.

  • X: Número de pixels no eixo X onde começa a área de assinatura.
  • Y: Número de pixels no eixo Y onde começa a área de assinatura.
  • Largura: Número de pixels que a área de assinatura se estende horizontalmente.
  • Altura: Número de pixels que a área de assinatura se estende verticalmente.
  • (Pré-visualização da página do PDF): O usuário deve clicar e arrastar o mouse sobre esta área para selecionar a área visível da assinatura. As posições e o tamanho da área de assinatura são expressos em pixels em relação à escala em que a imagem da página é exibida.
  • <<: Botão para mover a visualização para a primeira página do PDF.
  • <: Botão para mover a visualização para a página anterior do PDF. Pode-se usar a tecla "seta para a esquerda" do teclado para o mesmo efeito.
  • >: Botão para mover a visualização para a próxima página do PDF. Pode-se usar a tecla "seta para a direita" do teclado para o mesmo efeito. Se estiver na última página do PDF, esta opção permitirá criar uma página em branco ao final do documento para inserir a assinatura.
  • >>: Botão para mover a visualização para a última página do PDF.

Nesta mesma janela, na parte inferior, é possível indicar a página ou páginas onde a assinatura visível deve aparecer. As opções permitidas são:

  • Página atual: A assinatura visível aparecerá na página selecionada no seletor de posição superior.
  • Todas as páginas: A assinatura visível aparecerá em todas as páginas do documento.
  • Seleção de páginas: Nesta opção é possível selecionar a página ou intervalo de páginas onde a assinatura visível aparecerá. Para indicar uma página ou várias páginas, basta separar por vírgulas (1,4,7: Aparece nas páginas 1, 4 e 7) ou indicar um intervalo de páginas com um hífen (1-8: Aparece nas páginas de 1 a 8). Também é possível indicar páginas a partir do final do documento usando valores negativos (-2: Aparece na penúltima página). Esses mecanismos podem ser combinados. Por exemplo:
    • 3-6, 8, 10--1: Indica que aparecerá nas páginas de 3 a 6, na 8 e da 10 até a última página.

O botão com um ponto de interrogação (?) permite abrir a página de ajuda para assinatura visível.

A área de assinatura desaparece ao mudar de página do PDF. Selecione a área para a assinatura visível do PDF e, em seguida, clique no botão Próximo.

Em seguida, será exibida uma nova janela onde o usuário poderá configurar a aparência da assinatura.

Esta interface permite configurar as seguintes propriedades para a assinatura visível:

  • Pré-visualização. É possível configurar o arquivo de imagem clicando sobre a pré-visualização da assinatura. Não são permitidas imagens com transparência.
  • Configuração da assinatura:
    • Imagem da assinatura: Caminho da imagem com a rubrica da assinatura. Pode ser selecionada pelo botão Procurar e removida pelo botão Limpar. Imagens com transparência não são permitidas e serão ignoradas se encontradas.
    • Texto da assinatura: Texto a ser exibido na assinatura do PDF. Para configurar o texto a ser exibido na assinatura, é possível usar uma série de palavras-chave que serão substituídas pelos textos apropriados do titular ou emissor do certificado de assinatura no momento da assinatura. São elas:

      • $$SUBJECTCN$$
        • Nome comum (CN, Common Name) dentro do X.500 Principal do titular do certificado de assinatura.

      • $$ISSUERCN$$
        • Nome comum (CN, Common Name) dentro do X.500 Principal do emissor do certificado de assinatura.

      • $$CERTSERIAL$$
        • Número de série do certificado de assinatura.

      • $$SIGNDATE=PADRÃO$$
        • Data da assinatura, onde PADRÃO deve indicar o formato em que a data deve ser exibida, seguindo o esquema definido pela Oracle para a classe SimpleDateFormat.

      • $$GIVENNAME$$
        • Nome do titular (G, Given Name) declarado no certificado. Este valor pode não aparecer no certificado, caso em que o padrão será substituído por uma string vazia.

      • $$SURNAME$$
        • Sobrenome do titular (SN, Surname) declarado no certificado. Este valor pode não aparecer no certificado, caso em que o padrão será substituído por uma string vazia.

      • $$ORGANIZATION$$
        • Organização (O, Organization) do titular declarada no certificado. Este valor pode não aparecer no certificado, caso em que o padrão será substituído por uma string vazia.

      • $$PSEUDONYM$$
        • Pseudônimo do certificado. Este valor pode não aparecer no certificado, caso em que o padrão será substituído por uma string vazia.

      • $$OU$$
        • Unidade Organizacional (OU, Organization Unit) do certificado. Este valor pode não aparecer no certificado, caso em que o padrão será substituído por uma string vazia.

      • $$OUS$$
        • Lista de Unidades Organizacionais do certificado. Caso haja mais de uma, todas serão apresentadas separadas por ", ". O certificado pode não ter Unidades Organizacionais definidas, caso em que o padrão será substituído por uma string vazia.

      • $$TITLE$$
        • Informação sobre o cargo ou função do titular do certificado. Este valor pode não aparecer no certificado, caso em que o padrão será substituído por uma string vazia.

      • $$REASON$$
        • Razão pela qual o PDF é assinado, configurada na operação de assinatura pela propriedade "signatureReason". O padrão será substituído por uma string vazia se este valor não for definido.

      • $$LOCATION$$
        • Cidade onde o PDF é assinado, configurada na operação de assinatura pela propriedade "signatureProductionCity". O padrão será substituído por uma string vazia se este valor não for definido.

      • $$CONTACT$$
        • Informação de contato do signatário do PDF, configurada na operação de assinatura pela propriedade "signatureContact". O padrão será substituído por uma string vazia se este valor não for definido.

    • O texto pode ter as seguintes opções de formatação:
      • Fonte.
      • Tamanho da fonte.
      • Rotação do texto.
      • Estilo da fonte (Negrito, Itálico, Sublinhado ou Tachado).
      • Cor do texto.

      A caixa "Lembrar configuração" permite salvar a configuração estabelecida (texto, imagem e estilos) para uso nas próximas assinaturas. O botão "Restaurar configuração padrão" retorna a configuração do painel aos valores padrão.

    Se não desejar adicionar uma assinatura visível ao PDF, antes de clicar no botão assinar na tela de assinatura, certifique-se de que a caixa "Tornar a assinatura visível dentro do PDF" não está selecionada.


    Propriedades das marcas visíveis

    Ao habilitar a inserção de marcas visíveis no PDF, o usuário poderá selecionar uma área sobre o PDF que será assinado.

    Este menu permite usar o mouse para selecionar uma área retangular em uma ou todas as páginas do PDF.

    • Selecionar imagem: Ao clicar neste botão, será aberta uma janela onde é possível selecionar a imagem da marca. Imagens com transparência não são permitidas e serão ignoradas se encontradas.
    • X: Número de pixels no eixo X onde a marca será posicionada.
    • Y: Número de pixels no eixo Y onde a marca será posicionada.
    • Largura: Número de pixels que a marca deve se estender horizontalmente.
    • Altura: Número de pixels que a marca deve se estender verticalmente.
    • (Pré-visualização da página do PDF): O usuário deve clicar e arrastar o mouse sobre esta área para selecionar a área onde a marca será colocada.
    • <<: Botão para mover a visualização para a primeira página do PDF.
    • <: Botão para mover a visualização para a página anterior do PDF. Pode-se usar a tecla "seta para a esquerda" do teclado para o mesmo efeito.
    • >: Botão para mover a visualização para a próxima página do PDF. Pode-se usar a tecla "seta para a direita" do teclado para o mesmo efeito.
    • >>: Botão para mover a visualização para a última página do PDF.
    • Incluir imagem em todas as páginas: Caixa para selecionar que a marca seja inserida em todas as páginas do PDF.

    A área da marca desaparece ao mudar de página do PDF. Selecione a página, imagem e área para a marca visível no PDF e, em seguida, clique no botão Aceitar.

    Se não desejar adicionar uma marca visível ao PDF, antes de clicar no botão assinar na tela de assinatura, certifique-se de que a caixa "Inserir uma marca visível dentro do PDF" não está selecionada.
