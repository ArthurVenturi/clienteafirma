Tela de assinatura

Nesta tela você pode selecionar os documentos ou diretórios que deseja assinar.

Você pode selecionar um ou mais documentos clicando no botão "Selecionar arquivos para assinar" ou arrastando os documentos e/ou diretórios para a área indicada (o quadro de cor cinza escuro).

Ao carregar os documentos para assinar, serão exibidos alguns detalhes sobre eles, que variarão conforme seja selecionado um único arquivo ou vários:

No quadro habilitado para arrastar os documentos que deseja assinar, aparecerão as informações do(s) arquivo(s) selecionado(s).

Se você selecionou apenas um arquivo e possui um aplicativo padrão para esse tipo de documento, aparecerá o botão "Ver arquivo" para abri-lo. Além disso, serão exibidas informações relativas ao arquivo e à assinatura que será gerada (formato, atributos e opções de assinatura). Caso tenha configurado dados para o formato de assinatura selecionado que serão incluídos na própria assinatura, como a política de assinatura, será exibido um link "Ver todos..." para visualizar todos os atributos definidos.

Se foram selecionados vários arquivos ou diretórios, você pode dar duplo clique para abri-los ou clicar com o botão direito para ver as opções "Ver arquivo" e "Ver os atributos de assinatura".

O botão "Assinar" localizado na parte inferior da tela permitirá assinar o documento. Se o repositório de certificados do seu sistema operacional ainda não tiver terminado de carregar, o botão aparecerá desabilitado e com uma imagem animada de espera. Nesse caso, aguarde até que desapareça e o botão seja habilitado.

Após clicar no botão, o programa gerará uma assinatura para cada documento selecionado. Para cada tipo de documento, a assinatura será gerada com o formato definido no painel de preferências.

Caso deseje assinar em outro formato diferente o arquivo já selecionado, será possível modificar selecionando o link "(Alterar)" ao lado das informações sobre o formato. Ao selecioná-lo, será aberta a seguinte janela para escolher o formato desejado:

Os tipos de documento e os formatos de assinatura suportados são mostrados na tabela a seguir:

Ao assinar arquivos, será exibida uma janela para salvar a assinatura gerada se apenas um documento for selecionado ou um diálogo para selecionar o diretório de saída se vários forem selecionados. Após salvar as assinaturas, será exibida a tela de detalhes da assinatura.

O processo de assinatura pode ser alterado pelos plugins instalados no Autofirma. Consulte a seção de gestão de plugins para saber mais.

Além dos botões e elementos gráficos, você também pode usar a barra de menu ou os seguintes atalhos de teclado:

  • Control (⌘ no macOS) + S : Selecionar arquivo.
  • Control (⌘ no macOS) + F : Assinar o arquivo selecionado.

Configuração dos formatos de assinatura

Você pode configurar o formato de assinatura a ser utilizado para cada tipo de arquivo e as particularidades desses formatos, bem como o comportamento geral do aplicativo, a partir do painel de preferências disponível no menu Opções.

Alguns formatos ou opções de assinatura podem fazer aparecer opções de configuração adicionais na própria janela de assinatura:

  • Opções de assinatura PDF:

    A assinatura PDF de um único documento permite configurar para que a assinatura seja exibida no documento. Para isso, marque a caixa "Tornar a assinatura visível dentro do PDF".

    Também é possível inserir uma marca visível no PDF ao assinar. Esta opção modificará o PDF adicionando a imagem selecionada e, em seguida, o assinará. Para usar esta função, marque a caixa "Inserir marca visível no PDF". Esta caixa estará desabilitada se o documento PDF já estiver assinado, pois a inserção de uma marca invalidaria as assinaturas anteriores.

    Outra opção disponível é gerar um PDF certificado, possível se a assinatura a ser realizada for no formato PAdES e o documento não estiver previamente assinado. Isso é feito ao marcar a caixa "Gerar PDF certificado", e o nível de certificação selecionado será aplicado ao documento na lista de opções abaixo dessa caixa. A lista de opções estará desabilitada se a caixa "Gerar PDF certificado" não estiver habilitada.

    Essas opções aparecerão marcadas ou não por padrão conforme definido no painel de preferências. Ao selecionar qualquer uma dessas caixas, será exibido o painel de Propriedades de elementos visíveis em PDF.

  • Adicionar uma nova assinatura a um documento:

    Ao adicionar uma nova assinatura binária ou XML a um documento de assinatura, é possível configurar se a nova assinatura será feita sobre o próprio documento (coassinatura) ou sobre as assinaturas já introduzidas (contrafirma). Esta opção pode ser configurada no painel de preferências dos formatos CAdES e XAdES. Quando aplicável, o Autofirma mostrará ao usuário um botão "Opções Avançadas" para acessar diretamente essas opções de configuração.

Nos processos de assinatura de múltiplos documentos, qualquer configuração que obrigue a interromper o processo de assinatura será omitida. Assim, por exemplo, será omitida a solicitação de senha dos PDFs e a configuração de assinatura visível e marca visível será solicitada uma única vez e aplicada a todos os documentos com a mesma configuração.
