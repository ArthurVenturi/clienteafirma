O aplicativo Autofirma permite realizar facilmente assinaturas eletrônicas avançadas em arquivos locais.

Para simplificar o processo para o usuário, o formato de assinatura a ser utilizado é determinado automaticamente com base no arquivo de dados selecionado. O usuário pode modificar o formato de assinatura padrão para cada tipo de arquivo no menu de preferências.

Todas as assinaturas CAdES, XAdES e PAdES-BES geradas pelo "Autofirma" são realizadas conforme os perfis Baseline. Não é o caso das assinaturas PAdES Básicas.

O Autofirma foi projetado para utilizar o DNI eletrônico (DNIe) como dispositivo preferencial para a criação de assinaturas em sistemas com leitor de cartões inteligentes instalado, embora possa usar qualquer outro certificado instalado no sistema operacional. O Autofirma é compatível com DNIe 2.0 e DNIe 3.0, tanto por chip de contato quanto por NFC. Ao utilizar um DNIe, o certificado de assinatura é selecionado automaticamente.

O uso do Autofirma é muito simples, contando apenas com três telas em sua interface de usuário:

  • Tela de espera para inserção do DNIe (aparece apenas quando leitores de cartões inteligentes são detectados).
  • Tela de assinatura de arquivos.
  • Tela de resultado da assinatura.

Você pode acessar esta janela de ajuda a qualquer momento pressionando a tecla F1 (⌘+? no macOS).

Adequação ao Esquema Nacional de Segurança

Os produtos da Suite @firma podem conter, entre os algoritmos disponíveis, alguns não recomendados pelo Guia 807 do Esquema Nacional de Segurança (ENS; editado pelo Centro Criptológico Nacional, CCN) vigente no momento da publicação deste documento. Portanto, é responsabilidade dos aplicativos que utilizam esses produtos configurar adequadamente as chamadas para gerar o resultado esperado, válido e adequado para o momento e o nível de segurança desejado, utilizando para isso algoritmos da família SHA-2 conforme especifica essa norma para a geração de assinaturas eletrônicas.

Você pode consultar a norma vigente no seguinte link:
https://www.ccn-cert.cni.es/series-ccn-stic/800-guia-esquema-nacional-de-seguridad/513-ccn-stic-807-criptologia-de-empleo-en-el-ens/file.html
